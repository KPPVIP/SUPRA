Установка Add-on:

1. Создайте папку под названием "a80" по пути:\Grand Theft Auto V\mods\update\x64\dlcpacks
2. При помощи OpenIV — извлеките "dlc.rpf" по пути: \Grand Theft Auto V\mods\update\x64\dlcpacks

[Редактирование dlclist.xml]
3. При помощи OpenIV — перейдите по пути:\Grand Theft Auto V\update\update.rpf\common\data — и извлеките файл"dlclist.xml" в удобное для вас место
4. Добавьте в него новую строку: 

	<Item>dlcpacks:/a80/</Item>

5. Сохраните изменения и замените программой OpenIV — по пути:\Grand Theft Auto V\update\update.rpf\common\data

Спавнить по имени: a80
